# 📚 Preonboarding 2번째 기업 과제

## 🎀 작업 내역
* 기존의 자바스크립트 코드를 타입스크립트로 변환
* Jest로 API 호출 단위 테스트 구현
* 인풋 입력에 useDebounce 커스텀 훅 적용
* 결과 리스트에 로딩 spinner와 무한 스크롤 적용
* Figma 디자인 화면 컴포넌트 구현
* 검색 결과에 포함된 키워드 하이라이팅

## 💻 로컬 설치 및 실행방법

1. Clone this repo:

```bash
git clone ...
```

2. Install dependencies & packages

```bash
npm i
# OR
yarn
```

3. Run application

```bash
npm run start
# OR
yarn start
```
